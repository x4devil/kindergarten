Используем БД MySQL
Для задания сервера, логина и пароля в проекте открываем файл "App.config" и меняем соответствующие поля.
Если прога долго запускаеться, то идет подключение к БД и скорее всего будет ошибка, т.к. при верных данных подключается быстро.

Весь код комментируем, даже если это обработчики событий, которые генерит студия(ибо потом будете в ручную делать документацию).
Для каждого элемента на форме задает TabIndex. 
Элементы переименовываем (кроме Label), для остальных будем использовать правила:
	1) TextBox -> 'tbИмя'
	2) DataGrid -> 'dgИмя'
	3) ComboBox -> 'cbИмя'
	4) Button -> 'btnИмя'
	5) CheckBox -> 'cbИмя'
	6) DateTimepicker -> 'dtИмя'
Если обращаемся к элементам, то пишем  'this' (пример: this.btnCancel)
Все методы связанные с sql пишем в классе MySQLConnector
Глобальные переменные(доступные всем формам, а не во всей форме) храним в GlobalVars, но если понадобиться можете создать свои.
